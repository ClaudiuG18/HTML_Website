Offline Password Manager

My Offline Password Manager is a robust and secure application built using WPF and C#, with SQLite as the database backend. This tool is designed to keep your credentials safe and manage them efficiently.

Key features :

  • AES Encryption: All passwords are AES encrypted within the database, ensuring your sensitive information is protected against unauthorized access.
  • Multi-User Support: The application allows registration and management of multiple users, making it ideal for shared or family use.
  • Comprehensive Credential Management: Easily add and remove passwords and usernames. For each pair of username and password, you can also specify, add, or remove the corresponding application that uses these credentials.

Additional Highlights:

  • User-Friendly Interface: Built with WPF, the application offers a clean and intuitive interface for seamless interaction.
  • SQLite Database: Utilizes SQLite for efficient and reliable data storage, ensuring your information is always accessible when you need it.
  • Offline Access: No internet connection required. Your data stays on your device, offering peace of mind and eliminating the risk associated with online storage.
