Robo-Jack project

This project is a robot-arm that can be controlled via web in real time using:

  • Nginx used to serve the WebApp from a Raspi 4b
  • Websockets a protocol for full-duplex communication channels over a single TCP connection
  • Arduino R4 Wifi a microcontroler for controlling the servos and handling the incoming websockets messages
, WEBSOCKETS, . The robot is controlled by a Raspberry Pi that is connected to a PCA9685 servo controller. The robot has 4 wheels and 2 servos that control the direction of the robot. The robot can move forward, backward, left, right and stop. The robot can also rotate 360 degrees. The robot is controlled by a remote control that sends signals to the Raspberry Pi. The Raspberry Pi then processes the signals and sends them to the PCA9685 servo controller. The PCA9685 servo controller then controls the servos and the wheels of the robot. The robot is powered by a 12V battery and the Raspberry Pi is powered by a 5V power supply. The robot is made of 3D printed parts and is controlled by a Python script that runs on the Raspberry Pi.
